## 🚀 30인 실시간 멀티플레이어 모드

Firebase Realtime Database를 활용한 Kahoot 스타일 실시간 퀴즈

| 화면 | 설명 |
|---|---|
| 📺 `operator.html` | TV/PC용 운영자 화면 — 게임 선택, 타이머, 정답 공개 |
| 📱 `play.html` | 플레이어 폰 화면 — QR 스캔 후 A/B/C/D 답안 선택 |
| 🏆 `scoreboard.html` | 실시간 랭킹보드 (1점씩 카운트업 애니메이션, TV 모드) |

### 운영 방법
1. [Firebase Console](https://console.firebase.google.com) → 새 프로젝트 → Realtime Database → 테스트 모드로 생성
2. `operator.html`을 TV/PC에서 열고 Firebase URL 입력 → 저장
3. 게임 종류/문제 수/시간 설정 후 "새 세션 시작"
4. 화면의 QR 코드를 플레이어들이 스캔 → 닉네임 입력 → 자동 참가
5. "▶ 다음 문제" 클릭 → 모든 플레이어가 답하면 자동 공개!

## 📊 점수 시스템

- **개인 게임**: Firebase 연결 시 자동으로 실시간 랭킹에 반영
- **멀티플레이**: 정답 100점 + 속도 보너스 (1위 +50점, 상위 50% +20점)
- **TOP 10**: 상품 대상자로 표시 🎁
- **닉네임**: play.html에서 한 번 입력하면 모든 게임에 자동 적용
